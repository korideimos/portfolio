# Портфолио

<!-- Цель создания проекта, для какого курса, предмета выполнялся проект -->
### Проектное задание в рамках курса _Проектирование и разработка веб-сайтов_
<!--  Описание проекта, чему научилась, что попробовала реализовать в рамках создания проекта -->
> Стояла задача создать проект сайта и выполнить его реализацию, запусв локальный веб-сервер при помощи Apache
> 1. Разработать техническое задание, прототипы и дизайн-макеты страниц веб-сайта - [ссылка](https://www.figma.com/file/lIu82c4AWLHTMjsPUXuDOx/%D0%9F%D0%BE%D1%80%D1%82%D1%84%D0%BE%D0%BB%D0%B8%D0%BE?type=design&node-id=0%3A1&mode=design&t=39C5iyh91qin2do7-1)  на дизайн-макеты
> 2. В соответствии с дизайн-макетами выполнить верстку страниц веб-сайта.
> 3. Спроектировать и реализовать базу данных для проекта
> 4. Реализовать систему модерирования сайта
<!--  Какие технологии были использованы -->
### Технологии:
- __PHP__
- СУБД __MySQL__
- HTML и CSS
- _Для проекта: Figma, Miro, Notion_

<!--  Как использовать проект, какие особенности использования -->
Проект предназначен для сборки своего сайта портфолио, редактируемого под каждую конкретную вакансию - в базу данных необходимо ввести все данные о своем опыте, образовании, навыках, проектах
После ввода информации из всего этого можно собрать резюме+портфолио как конструктор, включив только те проекты(о портфолио), навыки и опыт(о резюме), которые инетересны работодателю в конкретной вакансии
